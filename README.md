# 🛍️ Shafa.ua Auto-Publisher & Scraper

Автоматизированная система для парсинга товаров и их публикации на украинском маркетплейсе **Shafa.ua**. 

Проект разработан для демонстрации навыков автоматизации работы со сложными, динамическими веб-интерфейсами (React/SPA), обработки асинхронных событий и обхода защитных/всплывающих механизмов сайта.

## 🚀 Ключевые технические решения

Этот бот — не просто набор кликов по XPATH. В нем реализованы продвинутые алгоритмы взаимодействия с современным вебом:

* **Smart React-Form Interaction:** Обход проблемы сброса состояния (state) в React-формах при программном вводе текста. Использование гибридного ввода (`pyperclip` + эмуляция реальной клавиатуры).
* **AI-Popup Handling:** Агрессивный перехват фокуса и программное закрытие непредсказуемых всплывающих окон (встроенного ИИ-генератора описаний Шафы), которые перекрывают DOM-элементы.
* **Intelligent DOM Analysis:**
  * Распознавание состояния кнопок (нажата/отжата) через динамический анализ длины CSS-классов соседей, что предотвращает конфликты со встроенными скриптами маркетплейса.
  * Умная очистка строковых ключей (обработка неразрывных пробелов `\xa0`, спецсимволов).
* **Robust Image Uploading:** Итеративная очередь загрузки фотографий. Решает проблему серверного "бутылочного горлышка" (bottleneck) при массовой загрузке тяжелых медиафайлов, предотвращая появление "битых" изображений на стороне сервера.
* **Clean Architecture:** Проект разделен на независимые модули (Scraper, Publisher, Models, Config), что делает код легко масштабируемым и поддерживаемым.

## 🛠️ Стек технологий

* **Язык:** Python
* **Web Automation:** Selenium WebDriver
* **Управление драйверами:** `webdriver-manager` (автоматическая установка актуальных бинарников Chrome)
* **Взаимодействие с ОС:** `pyperclip` (работа с системным буфером обмена для надежной вставки объемных текстов)

```text

│   ├── scraper/         # Логика сбора данных (картинки, характеристики)
│   ├── publisher/       # Логика интеллектуальной публикации объявлений

Клонируйте репозиторий:
