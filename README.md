Автоматизированная система полного цикла для парсинга товаров, их интеллектуальной публикации и управления статусом на украинском маркетплейсе **Shafa.ua** с современным графическим интерфейсом.

Проект разработан для демонстрации навыков автоматизации работы со сложными, динамическими веб-интерфейсами (React/SPA), работы в многопоточном режиме с GUI, обработки асинхронных событий, обеспечения обратной совместимости данных и обхода защитных механизмов сайта.

## 🚀 Ключевые технические решения

Этот бот — не просто набор кликов по XPATH. В нем реализованы продвинутые алгоритмы взаимодействия с современным вебом:

* **Modern GUI (CustomTkinter):** Удобный графический интерфейс с поддержкой ввода списка URL-адресов, выбора настроек (деактивация оригинала, автоматическая публикация) и встроенным окном логов в реальном времени.
* **Multithreading:** Запуск процесса автоматизации в отдельном рабочем потоке (`threading`), что предотвращает "замораживание" графического интерфейса во время работы веб-драйверов.
* **Multi-Tab Flow Control:** Бот умеет работать в многовкладочном режиме. Для каждого нового товара открывается отдельная вкладка. По окончании процесса заполнения, вкладка остается открытой (если отключена автопубликация), и бот переключается обратно на рабочий процесс для следующего товара.
* **Full-Cycle Automation:** Полная автоматизация жизненного цикла объявления. Бот собирает данные, заполняет все поля новой формы и может автоматически деактивировать старый оригинал через временную вкладку.
* **Smart React-Form Interaction:**
  * Обход проблемы сброса состояния (state) в React-формах при программном вводе. Использование гибридного ввода (`pyperclip` + эмуляция клавиатуры) для защиты от потери символов.
  * Безопасное взаимодействие с Combobox-элементами (мульти-выбор): последовательный ввод тегов без стирания предыдущих значений.
* **Intelligent DOM Analysis & Navigation:**
  * Поиск элементов на основе структуры (например, по тексту соседних плейсхолдеров), что делает код устойчивым к динамическому изменению CSS-классов и ID элементов в React.
  * Распознавание состояния кнопок (нажата/отжата) через динамический анализ длины CSS-классов соседей, предотвращающее конфликты со встроенными скриптами маркетплейса.
  * Взаимодействие со скрытыми элементами (автоматическое раскрытие кнопок "Читати далі" для полного парсинга DOM).
* **Data Normalization & Backward Compatibility:** Алгоритмы "на лету" адаптируют устаревшие форматы данных (например, парсинг сырой строки `36/S/44` для извлечения международного размера) под новые требования формы публикации.
* **AI-Popup & Responsive Handling:**
  * Агрессивный перехват фокуса и программное закрытие непредсказуемых всплывающих окон (встроенного ИИ-генератора описаний Шафы).
  * Форсирование десктопного Viewport (`--start-maximized`) для обхода адаптивной верстки, скрывающей важные элементы управления в мобильном виде.
* **Robust Image Uploading:** Итеративная очередь загрузки фотографий с программной сортировкой массива. Решает проблему асинхронного серверного "бутылочного горлышка" (bottleneck), гарантируя, что главная фотография всегда отрендерится первой.

## 🛠️ Стек технологий

* **Язык:** Python
* **Web Automation:** Selenium WebDriver
* **GUI Library:** CustomTkinter
* **Управление драйверами:** `webdriver-manager` (автоматическая установка актуальных бинарников Chrome)
* **Взаимодействие с ОС:** `pyperclip` (работа с системным буфером обмена для надежной вставки объемных текстов)

## 📂 Структура проекта

```text
shafa-parser/
├── src/
│   ├── scraper/         # Логика сбора данных (картинки, характеристики, раскрытие текста)
│   ├── publisher/       # Логика интеллектуальной публикации и деактивации объявлений
│   ├── models.py        # Дата-классы (Product) для строгой типизации
│   ├── config.py        # Настройки путей и профиля браузера
│   └── utils/           # Вспомогательные функции (логгер, работа с ФС, обработка картинок)
├── main.py              # Точка входа в приложение (GUI на CustomTkinter)
├── .gitignore           # Игнорирование кэша, логов и локальных БД
└── requirements.txt     # Зависимости проекта
```

## ⚙️ Установка и запуск

Клонируйте репозиторий:
```bash
git clone https://github.com/DanilTet/shafa-parser.git
cd shafa-parser
```

Установите зависимости:
```bash
pip install -r requirements.txt
```

Запустите приложение:
```bash
python main.py
```
